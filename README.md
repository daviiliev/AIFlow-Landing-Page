# 🚀 AIFlow – Premium Startup Landing Page

A modern, premium landing page for an AI startup. Built with **Vanilla HTML, CSS & JavaScript** – no frameworks, just clean code.

![Desktop View](screenshots/desktop.png)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌙 **Dark Theme** | Modern, edgy design with deep colors |
| ✨ **Animated Background** | Glowing gradient effects that move |
| 🪞 **Glassmorphism** | Frosted glass effect on cards |
| 📱 **Fully Responsive** | Perfect on mobile, tablet, and desktop |
| 🎯 **Interactive Elements** | Smooth animations & hover effects |
| 💰 **Pricing Section** | Professional tier display with "Popular" badge |
| 🖼️ **Image Placeholders** | Ready for your visuals |
| ⚡ **Fast Loading** | Lighthouse score 95+ |
